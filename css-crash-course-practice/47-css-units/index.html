Working with Units in CSS

Types of Units in CSS

  1. Absolute Units: (fixed)
    • px (pixels)
    • in (inches)
    • cm (centimeters)
    • mm (millimeters)
    • pt (points)
    • pc(picas)
  2. Relative Units:
    • em
    • rem (root em)
    • % (percentage)
    • vw (viewport width)
    • vh (viewport height)
